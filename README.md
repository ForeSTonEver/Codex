# 파이썬 코딩 수행평가 프로그램

온라인 저지(Online Judge) 방식의 파이썬 코딩 수행평가 시스템입니다.

## 📋 프로그램 개요

- **목적**: 파이썬 기초 개념 학습 및 수행평가
- **문제 구성**: 상, 중, 하 각 1문제 (총 3문제)
- **포함 요소**: 변수, 자료형, 선택구조, 반복구조, 리스트

## 🎯 문제 설명

### 하 (쉬움) - 리스트에서 최댓값과 최솟값 찾기
**포함 요소**: 변수, 자료형(int, list), for문, if문

정수 리스트가 주어질 때:
1. 최댓값 구하기
2. 최솟값 구하기
3. 두 값의 합 계산

**입력 예시**:
```
5
10 20 5 30 15
```

**출력 예시**:
```
30
5
35
```

---

### 중 (중간) - 학점 계산 및 성적 순위
**포함 요소**: 변수, 자료형(int, str, float, list), for문, if/elif/else, sorted()

학생들의 점수가 주어질 때:
1. 각 학생의 학점 계산 (A: 90~, B: 80~, C: 70~, D: 60~, F: ~59)
2. A학점 개수 세기
3. 평균 점수 계산 (소수점 1자리)
4. 점수를 내림차순으로 정렬

**입력 예시**:
```
5
95
82
78
91
65
```

**출력 예시**:
```
2
82.2
95 91 82 78 65
```

---

### 상 (어려움) - 소수 판별 및 필터링
**포함 요소**: 변수, 자료형(int, bool, list), 중첩 for문, if문, 함수

정수 리스트가 주어질 때:
1. 소수 판별
2. 소수의 개수 세기
3. 소수들의 합 계산
4. 각 수별 소수 여부를 True/False로 표현

**입력 예시**:
```
6
2 3 4 5 6 7
```

**출력 예시**:
```
4
17
True True False True False True
```

## 💡 코딩 팁

### 하 (쉬움)
```python
# 리스트에서 최댓값 찾기
max_val = numbers[0]
for num in numbers:
    if num > max_val:
        max_val = num
```

### 중 (중간)
```python
# 조건에 따른 학점 부여
if score >= 90:
    grade = 'A'
elif score >= 80:
    grade = 'B'
# ...
```

### 상 (어려움)
```python
# 소수 판별 함수
def is_prime(n):
    if n < 2:
        return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            return False
    return True
```
